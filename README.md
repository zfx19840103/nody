# Vue 3 + TypeScript + Vite

vite+vue3+ts

## 安装

```bash
git clone https://github.com/zhangfangxu/vite-project-nody.git <project-name>
cd vite-project-nody
npm install or yarn
```

## 运行

```bash
npm run dev or yarn dev
``` 
## 打包

```bash
npm run build or yarn build
``` 
## 预览运行打包文件dist

```bash
npm run preview or yarn preview
```
